### 简单日志封装, 根据"日志尺寸"和"日期"两个维度自动分割日志
1. 对于非结构化输出, 日志使用方式上和使用fmt包类似, 都是使用占位符.
2. 对于结构化输出, 请使用带"w"结尾的函数, 例如:"Infow", 函数签名为:Infow(msg,key,value).
   - 对于使用结构化输出时, value需要注意的3个规则.
     1. 带json tag的 结构体 / 对象, 可直接当做value传入.
     2. json.Marshal 产生的 []byte, 需要使用 json.RawMessage() 转换.
     3. 纯字符串 / 表单 / 原始文本, 使用string() 转换
